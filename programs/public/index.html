Sabo HTTP Server

A web server written in a stack-based language. Yes, really.

Try these endpoints:

/hello — plain text greeting
/greet/:name — parameterized greeting
/api/echo?msg=...&from=... — echo query params as JSON
/api/time — current server time
/api/fib/:n — compute fibonacci
/api/stats — server stats (request count)

POST endpoints:

curl -X POST -d '{"text":"hello world"}' localhost:8080/api/reverse
curl -X POST -d '{"items":[3,1,4,1,5]}' localhost:8080/api/sort
